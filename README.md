# Search Image Application - MVVM

using https://unsplash.com/documentation API to get Images and display it in my application .

I'm using in this project :

* API
* Coroutines
* Navigation Component
* Gson
* Pagin 3
* Dagger Hilt
* MVVM architecture
* View Model
* Retrofit
* Glide
